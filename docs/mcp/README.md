# MCP Setup Guides

This directory contains setup guides for integrating ws-ctx-engine MCP server with various IDEs and editors.

## Available Guides

### Qoder IDE
- **Guide:** [Qoder MCP Setup Guide](setup-guide/qoder/README.md)
- **Sample Config:** [mcp.json.example](setup-guide/qoder/mcp.json.example)

#### Quick Start for Qoder
1. Copy `setup-guide/qoder/mcp.json.example` to your Qoder MCP config location
2. Adjust the workspace path if needed
3. Run `wsctx init` to initialize the engine
4. Restart Qoder
